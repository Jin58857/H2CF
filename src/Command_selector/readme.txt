任务2：训练带有元命令的执行网络
场景：假设敌方静止不动，在6v3的场景下，给每一个智能体的观测加入元命令，分别包括动作和目标。
智能体根据自己的观测执行相应的元命令。
需要解决的问题：
	1.我方控制6个智能体，每一个智能体在自己的观测基础上加入元命令。
	2.每一个智能体在做决策之前，将状态输入到一个注意力网络，重点关注和自己元命令相同的智能体的信息
	3.再加入遵循元命令的奖励。
预期效果：
	给智能体任意一个元命令，智能体可以完成，并且出现相同元命令的智能体之间的合作效果
	目前的元命令的动作：攻击，躲避
	攻击作用：靠近某个敌方，进行攻击
	躲避作用：不靠近某个敌方，攻击其他的智能体
需要研究思考的点：
	1.在现在的代码基础上，如何加入注意力网络？
	2.如何将元命令集成到智能体观测中？
	3.加入元命令之后的奖励该如何设置？
